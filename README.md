# 📊 Streaming de Dados em Tempo Real com Python

Este projeto simula um fluxo de dados em tempo real, representando a chegada contínua de pedidos e seu processamento imediato.

A proposta é demonstrar, de forma prática, conceitos fundamentais de Engenharia de Dados utilizando Python.

---

## 🚀 Visão Geral

O sistema gera pedidos automaticamente e os processa em tempo real, simulando um cenário semelhante a aplicações de delivery ou plataformas digitais.

Cada novo dado é tratado no momento em que chega, permitindo acompanhar métricas de forma contínua.

---

## ⚙️ Funcionalidades

- Geração de pedidos aleatórios
- Simulação de streaming de dados
- Processamento contínuo em tempo real
- Atualização dinâmica das informações

---

## 🧠 Conceitos de Engenharia de Dados Aplicados

- **Fontes de dados** → geração simulada de dados
- **Streaming de dados** → processamento contínuo
- **Processamento em tempo real** → atualização imediata
- **Estrutura de dados** → uso de dicionários para representação

---

## 🛠️ Tecnologias Utilizadas

- Python
- Bibliotecas padrão:
  - `random` → geração de dados
  - `time` → controle de fluxo
